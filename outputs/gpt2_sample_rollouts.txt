HOW TO READ THIS:
  Each scenario shows a model-sampled sequence of predicted incidents.
  The model is given an 8-token context (time, weather, era) and
  autoregressively generates realistic incident sequences.
  Each row = one predicted incident event.

KEY FIELDS FOR THE MAP:
  Ward     -> polygon on the map (GeoJSON ward boundary)
  Station  -> responding fire station
  Type     -> colour/icon on map (Dwelling Fire=red, Outdoor=orange,
              False Alarm=grey, Special Service=blue)
  Gap      -> time since previous incident (useful for animation/timeline)

  5. Coordinate reference: WGS84 (standard lat/lon)
